rem

The rem is a unit of length that is equal to the computed font size of the root element. It is used in typography to define the size of a font. The rem is a relative unit of length, which means that the size of a rem is not fixed and can vary depending on the font size of the root element. The rem is commonly used in web design to define the size of text and other elements on a web page.

This is a representation of a width resolution of a current screen (which is a FHD) so 120rem (which is 1920px), represented in rem.

This is a representation of a height resolution of a current screen so 67.5rem (1920px = 67.5rem), represented in rem.
