vw - Viewport Width

The vw is a unit of length that is equal to 1% of the width of the viewport. It is used in web design to define the size of elements on a web page relative to the width of the viewport. The vw is a relative unit of length, which means that the size of a vw is not fixed and can vary depending on the width of the viewport. The vw is commonly used in responsive web design to create layouts that adapt to different screen sizes.

This is a representation of a width resolution of a current screen (which is a FHD) so 100vw (which is 1920px), represented in vw.

This is a representation of a height resolution of a current screen so 55.5vw (1920px = 55.5vw), represented in vw.
