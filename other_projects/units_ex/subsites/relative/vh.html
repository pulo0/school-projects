vh - Viewport Height

The vh is a unit of length that is equal to 1% of the height of the viewport. It is used in web design to define the size of elements on a web page relative to the height of the viewport. The vh is a relative unit of length, which means that the size of a vh is not fixed and can vary depending on the height of the viewport. The vh is commonly used in responsive web design to create layouts that adapt to different screen sizes.

This is a representation of a width resolution of a current screen (which is a FHD) so 210.5vh (which is 1920px), represented in vh.

This is a representation of a height resolution of a current screen so 118.5vh (1920px = 118.5vh), represented in vh.
