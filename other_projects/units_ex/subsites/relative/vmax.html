vmax

The vmax is a unit of length that is equal to the larger of the viewport's height or width. It is used in web design to define the size of elements relative to the viewport. The vmax is a relative unit of length, which means that the size of a vmax is not fixed and can vary depending on the size of the viewport. The vmax is commonly used in web design to define the size of text and other elements on a web page.

This is a representation of a width resolution of a current screen (which is a FHD) so 100vmax (which is 1920px), represented in vmax.

This is a representation of a height resolution of a current screen so 55.5vmax (1920px = 55.5vmax), represented in vmax.
