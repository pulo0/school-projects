ch

The ch is a unit of length that is equal to the width of the digit '0' in the current font size. It is used in typography to define the size of a font. The ch is a relative unit of length, which means that the size of a ch is not fixed and can vary depending on the font size of the parent element. The ch is commonly used in web design to define the size of text and other elements on a web page.

This is a representation of a width resolution of a current screen (which is a FHD) so 228.6ch (which is 1920px), represented in ch.

This is a representation of a height resolution of a current screen so 128.6ch (1920px = 128.6ch), represented in ch.
