vmin

The vmin is a unit of length that is equal to the smaller of the viewport's height or width. It is used in web design to define the size of elements relative to the viewport. The vmin is a relative unit of length, which means that the size of a vmin is not fixed and can vary depending on the size of the viewport. The vmin is commonly used in web design to define the size of text and other elements on a web page.

This is a representation of a width resolution of a current screen (which is a FHD) so 210.5vmin (which is 1920px), represented in vmin.
